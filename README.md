# decomoji-color-picker

デコモジファイルからデコモジカラーを引き当てるスクリプト。

## Example

```
node src/index.js --input-dir example
```

## Arguments

### `--input-dir`
デコモジ PNG が格納されているディレクトリ。

default: `.`

### `--output`
出力ファイル。

default: `./dist/result.tsv`
